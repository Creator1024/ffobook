| 物品名    | 物品描述                                                     |
| --------- | ------------------------------------------------------------ |
| 圆木盾    | 用木头制作成的盾牌，防御力很低。在更多的时候是种累赘。但是性命珍贵。是以使用者却不少。 |

| 混银盾    | 用白银和秘银混合制造的盾牌，秘银的轻盈和坚硬使它成为最受欢迎的盾牌之一。 |
| 狼魂大盾  | 面积很大的盾牌，上面刻有一只狰狞的狼头，最大优点是覆盖面广，对无差别攻击的防御非常优良。 |
| 白虎之盾  | 以放弃灵活性来换取的绝对防御能力，在阵战中能发挥相当大的作用，天生神力者才能使用。 |

| 巨魔骨盾  | 极西之地的英雄杀死强大的巨魔之后从少数巨魔首领和勇士手里得到的盾牌。坚固无比。 |
| 北辰悬照  | 上面用星尘沙镶嵌着北斗星的盾牌，据说持此盾者能获星辰之力的加持，在战斗中占尽优势。 |
| 圣盾·焰灵 | 赤红的盾牌，上面封印有强大的火元素之灵。拥有极高的防御力。并且能够赋予使用者无比的勇气。 |
| 古铁盾    | 这面厚重的盾牌经历过了长久的岁月，虽然人人都称之为“古旧的铁物”，但是它的防御能力仍是相当出色的。 |
| 坚钢盾    | 将一整块优良的钢材锤炼成较薄的钢板，再将钢板反复曲卷成盾牌的大小，在每一层的中间都填入了吸收冲击的材料，最后经过加固和整形的工序。这样制作出来的盾牌具有十分强的抵御能力，不过也相当的重。 |
| 墨玉盾    | 这面盾牌是从一大块黑色的玉石中雕琢出来的，这样的工艺已经失传了。 |
| 月辉盾    | 这面盾牌上用淡蓝色宝石镶镶成了月亮的形状，它对魔法攻击的防御力要优于一般的盾牌。 |
| 刃牙盾    | 如同犬牙般的锐利弯刺布满了这面圆盾的表面，任何攻击持盾者的人都会因为这些弯刺而受伤。 |
| 符文盾    | 这面盾牌在铸作完成之后经过一系列的增强抗魔法能力的处理，以致它的表面布满了奇怪的符文。这种特化处理也使得它几乎对无力攻击毫无抵抗力。 |

| 长城      | 它的外观被塑造成微缩版的城墙一般，而正如名字所表示的，它不仅代表着一座古代的伟大建筑物，也代表着一个民族在抵御外敌时展示出来的意志。见证了无数朝代的兴起与衰败，这面盾牌所历练过的时间已经无从算起，它是所有最古老的盾牌的其中之一。 |

|  古器·幻仙  | 位于杖顶的浑圆宝石像是有生命的一样散发出一明一暗的光芒。古代的纪录有详细的记载着这种法杖的制作方式，但是其中最难的部分就是必须要一位仙人牺牲自己的元神方可炼成杖顶的仙灵石。这种魔杖在世上不过寥寥数把，也许每一把的后面都藏着惊心动魄的故事吧。 |
| 古器·轰雷神 | 传说可以呼唤雷神的魔杖，有着一击摧毁整个村落的纪录。但是随着古代王国与大灾难一同逝去，如何引出轰雷神真正威力的方法也一并失传了。 |
